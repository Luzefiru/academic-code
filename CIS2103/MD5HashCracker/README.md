# Usage

You can change the `CHARSET`, `MIN_LENGTH`, `MAX_LENGTH`, the online GitHub .txt word list, and the Rainbow Table file name in `MD5HashCracker.java`.

```bash
javac MD5HashCracker.java && time java MD5HashCracker [MD5_HASH]
```
